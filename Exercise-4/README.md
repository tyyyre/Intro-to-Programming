# Exercise 4 – Primitive Quiz

This exercise required writing a simple quiz program that checks a user's answer and provides feedback.

## Basic Requirements
- Ask the user: "What is the capital of France?"
- Check if the answer is correct.
- Print either "Answer is correct!" or "Answer is wrong!"

## Advanced Requirements
- Accept answers regardless of capitalization.
- Extend the quiz to include additional questions (e.g., 10 European capitals).
- Provide feedback for each question.

## Files
- primitive_quiz.py – basic version
- primitive_quiz_advanced.py – advanced version
