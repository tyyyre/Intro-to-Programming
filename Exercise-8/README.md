# Exercise 8 – Simple Search

This exercise demonstrates basic search functionality using lists and conditionals.

## Requirements
- Create a list of six names.
- Search the list for the name "Sam".
- Print whether the name is found or not.

## Optional Requirements
- Allow the user to input a search term instead of using a fixed name.

## Files
- simple_search.py – final solution
- simple_search_optional.py – optional version (user-input supported)
