# Exercise 5 – Days of the Month

This exercise uses a dictionary to map month numbers (1–12) to the number of days in each month.

## Basic Requirements
- Create a dictionary of month-to-days mappings.
- Ask the user for a month number.
- Validate the input and display the correct number of days.

## Advanced Requirement
- Ask for the year when February is selected.
- Adjust days based on leap year rules.

## Files
- days_of_the_month.py – basic version
- days_of_the_month_advanced.py – advanced version
