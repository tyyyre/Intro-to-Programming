# Exercise 10 – Is It Even?

This exercise determines whether a user-entered number is even or odd.

## Program Flow
1. `main()` asks the user for a number.
2. Number is passed to `check_even_odd()` function.
3. Function checks:
   - Returns "Even" if divisible by 2
   - Returns "Odd" otherwise
4. Result is printed from `main()`
