# Exercise 6 – Brute Force Attack

This program simulates a password entry system.

## Basic Requirements
- Define the correct password.
- Use a while loop to repeatedly ask for the password.
- Continue until the user enters 12345.

## Optional Requirements
- Allow only 5 attempts.
- Inform the user of remaining attempts.
- Deny access after the final attempt.

## Files
- brute_force.py – basic version
- brute_force_limited.py – optional requirements version
